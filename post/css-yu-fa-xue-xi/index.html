CSS语法

基本规则

CSS主要由两部分组成：选择器(selector) & 声明(declarations)

/* selector {property: value} */
h1 {color:red; font-size:14px;}

样式表书写

目前，主要采用内联样式表的书写方案

<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
  	<h1 style="color:red; font-size:14px;"></h1>
  </body>
</html>

@规则

通过@开头，后接标识符 目前用的比较多的是 @keyframes，通过定义关键帧实现动画效果

/* @keyframes */
@keyframes slidein {
    from {
        margin-left: 100%;
        width: 300%;
    }

    to {
        margin-left: 0%;
        width: 100%;
    }
}
